# todolist
# How compile
  ``` go build ./cmd/todo```
  
# How use

  Create task:
    ```./todo -add New task``` </br>

  Mark as complete
    ```./todo -complete=IndexOfTask``` </br>

  See list all tasks
    ```./todo -list``` </br>

  Delete task:
    ```./todo -del=IndexOfTask``` </br>
